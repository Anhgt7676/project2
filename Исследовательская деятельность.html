Это мой проект по элективному курсу проектной деятельности. Хочу представить сайт по достопримечательностям Тюмени. "

Данный проект я выбрал для того, чтобы лучше узнать свой город. Раннее у меня уже был опыт с сайтами, но я хочу продолжить этим заниматься.

Сайт (от англ. Site – место, позиция) – совокупность страниц, объединённых одной общей темой, дизайном, имеющих взаимосвязанную систему ссылок, расположенных в сети Интернет.<

Каждый сайт создаётся с какой-то определённой целью. Прежде всего, они нужны для передачи определённой информации пользователю сети. Назначение и внешний облик сайтов поражает своим разнообразием.

Существует много бесплатных конструкторов для создания сайта, такие как Tilda или Figma, которые сильно упрощают работу над сайтом. Я же создал сайт с помощью VS Code и языков HTML, CSS и Java.
